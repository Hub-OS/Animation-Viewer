# Animation Viewer

A utility for viewing BoomSheets animations. [Live Page](https://hub-os.github.io/Animation-Viewer/).

Not affiliated with [BoomSheets](https://store.steampowered.com/app/2189000/BoomSheets/).

## Building

```
npm install
npm run licenses
npm run build
```

A `dist` folder will be generated.

## Deploying to GitHub Pages

After building:

```
npm run deploy
```
