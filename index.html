Animation Viewer
GitHub

A utility for viewing BoomSheets animation files.

Not affiliated with BoomSheets.

Usage:

Drag and drop BoomSheets `.animation` and `.png` files with matching names into the dashed box to add them to the scene. Dropping files with the same name again will reload animations for existing sprites.

You can reorganize sprites in the dashed list to create attachments and adjust the draw order.

Any files containing the word "background" or ending in "_bg" will default to the BG playback mode.
